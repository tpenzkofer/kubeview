# kubeview has moved → tankertop

**This project was renamed and now lives at
[github.com/tankertop/tankertop](https://github.com/tankertop/tankertop).**

All development continues there — please update your bookmarks and installs:

```sh
# repoint an existing clone
git remote set-url origin https://github.com/tankertop/tankertop.git

# or install fresh
go install github.com/tankertop/tankertop@latest
```

The name changed because the tool now monitors both Kubernetes **and** Docker,
so it is no longer kube-specific. This repository is kept only as a redirect.
